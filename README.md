# Little Lemon API

## Endpoints

- **User registration**  
  `POST /api/registration/`

- **Login**  
  `POST /api/login/`

- **Menu items**  
  `GET  /api/menu/`  
  `POST /api/menu/`  

- **Table bookings**  
  `GET    /api/bookings/`  
  `POST   /api/bookings/`  
  `PUT    /api/bookings/{id}/`  
  `DELETE /api/bookings/{id}/`
